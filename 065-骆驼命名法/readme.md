#骆驼命名法

##题目描述
　　从C/C++转到Java的程序员，一开始最不习惯的就是变量命名方式的改变。C语言风格使用下划线分隔多个单词，例如“hello_world”；而Java则采用一种叫骆驼命名法的规则：除首个单词以外，所有单词的首字母大写，例如“helloWorld”。<br>
　　请你帮可怜的程序员们自动转换变量名。

##输入描述:
　　输入包含多组数据。<br>
　　每组数据一行，包含一个C语言风格的变量名。每个变量名长度不超过100。


##输出描述:
　　对应每一组数据，输出变量名相应的骆驼命名法。

##输入例子:
```
hello_world
nice_to_meet_you
```

##输出例子:
```
helloWorld
niceToMeetYou
```
